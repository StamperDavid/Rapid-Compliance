3. Predictive score calculated (Hot/Warm/Cold badge)

5. Activity timeline tracks all interactions
6. Next best action updates based on engagement

### **Deal Management Flow:**
1. Deal created ‚Üí Health score calculated
2. Detail page shows: Health 67/100 - AT RISK
3. Warnings: "No activity in 12 days", "Delayed in Proposal stage"
4. Recommendations: "Schedule follow-up call TODAY"
5. Activity timeline shows all touchpoints
6. Stakeholder map shows buying committee

### **Email Campaign Flow:**
1. Build email in drag-drop builder
2. Set up A/B test (test 2 subjects)
3. Send to 200 leads (100 each variant)
4. Dashboard shows: Variant B wins (52% open vs 45%)
5. Automatically sends remaining with winning variant
6. Inbox sync captures replies
7. AI processes and responds

### **Team Collaboration Flow:**
1. Rep logs activity on deal
2. Adds note with @mention to manager
3. Manager gets email notification
4. Manager assigns task to rep: "Send proposal by Friday"
5. Task appears on rep's kanban board
6. Rep completes task ‚Üí Deal health improves
7. Leaderboard updates with points
